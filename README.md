### Usage

Install dependencies:
```bash
python3 -m pip install requirements.txt
```

Training:
```bash
python3 analysis_tap_hold.py
```

Inferencing:
```bash
python3 inferencing.py
```

Mock inferencing (For testing in VR): 
```bash
python3 inferencing_mock.py
```
